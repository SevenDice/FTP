Available commands and arguments for them:

	ls		Shows list of all files and directories on current level. 
			Shows date, month, year or time, size and name of file or directory as default.
		-ms	Month
		-ds	Day
		-tys	Time or year
		-s	Size
		-on	Owner name
		-rc	Count of references
		-n	Name
		-gn	Group name
		-tm	File type and rights
	cat		Shows file content
	dl [fn][p]	Download file fn from current directory to directory p on your computer.
	cd [d]		Changes current directory to d (/../ to return to a higher level)
	exit		Exit client
